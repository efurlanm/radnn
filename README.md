# RADNN

*Last edited: 2024-12-16*

This repository curates my research on applying physics-based machine learning (PIML) to scientific problems. The '[src](src)' directory contains the main files. The '[docs](docs)' directory contains the [documentation](https://efurlanm.github.io/radnn) which attempts to provide an overview of the contents, as well as some personal notes I've made over time. As this research is ongoing, expect the repository to remain a work in progress (WIP) and subject to change.
