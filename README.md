# RADNN

*Last edited: 2024-12-31*

This repository contains my personal notes on research on applying physics-based machine learning (PIML) to scientific problems. The '[src](src)' directory contains the main files related to the research, as well as the documentation sources. The '[docs](docs)' directory contains the online documentation automatically generated using Markdown files and is available at <https://efurlanm.github.io/radnn>. As this research is ongoing, expect the repository to remain a work in progress (WIP) and subject to change.
