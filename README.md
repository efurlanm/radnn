# RADNN

*Last edited: 2024-12-21*

This repository curates my research on applying physics-based machine learning (PIML) to scientific problems. The '[src](src)' directory contains the main research files, and the documentation sources. The '[docs](docs)' directory contains the [documentation](https://efurlanm.github.io/radnn), which is an attempt to provide an overview of the content, as well as includes some personal notes I've made over time. As this research is ongoing, expect the repository to remain a work in progress (WIP) and subject to change.
